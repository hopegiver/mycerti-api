홈빌더 솔루션 요구사항 명세서 (업데이트)

1. 개요

    본 시스템은 회원이 다수의 홈페이지를 생성·편집·배포할 수 있는 홈빌더 솔루션을 제공한다.

    홈페이지는 위지윅(WYSIWYG) 기반 편집기를 통해 손쉽게 제작 가능하며, 퍼블리싱 시 정적 파일을 S3에 저장하고 CSR(Client Side Rendering) + API 연동 방식으로 서비스된다.

    관리자는 회원별 홈페이지 생성 개수를 설정 및 변경 가능하다.

    관리자 페이지는 2계층 구조를 가진다:

    회사 전체 관리자 페이지 (Super Admin)

    각 홈페이지별 관리자 페이지 (Site Admin)

2. 사용자 유형

    회원(User)

    다수의 홈페이지 생성 가능

    템플릿 선택 및 커스터마이징

    편집 → 저장 → 퍼블리싱 사이클 수행

    홈페이지 관리자(Site Admin)

    각 홈페이지별 전용 관리 페이지 제공

    콘텐츠(게시글, 상품, 폼 데이터 등) 관리

    방문자 통계 및 설정 관리 (사이트 제목, 로고, 메뉴, SEO 설정 등)

    회원(홈페이지 이용자) 관리 가능 (필요시)

    회사 관리자(Super Admin)

    전체 회원 및 홈페이지 관리

    회원별 홈페이지 수 한도 설정/변경

    템플릿 등록/관리 (AI 자동 생성 포함)

    전체 시스템 로그, 통계, 모니터링

    리소스 사용량(스토리지, 트래픽) 관리

3. 주요 기능 요구사항
3.1 회원 관리

    회원 가입/로그인 (이메일, 소셜 로그인 가능)

    회원별 홈페이지 생성 가능 수량 제한

    결제 플랜 연동 (기본 무료, 유료 업그레이드 가능)

3.2 홈페이지 생성 및 편집

    템플릿 기반 생성 및 수정 (AI 기반 자동 생성 포함)

    위지윅 편집기 (텍스트, 이미지, 비디오, 위젯, 섹션)

    반응형 지원

    저장 및 퍼블리싱 (S3 업로드, CDN 배포)

3.3 퍼블리싱 및 배포

    전체 퍼블리싱 및 부분 퍼블리싱 지원

    CSR 구조 + API 연동 방식

    도메인 연결 (기본 서브도메인 + 사용자 도메인)

    퍼블리싱 이력 관리

3.4 회사 전체 관리자 페이지 (Super Admin)

    회원 관리: 가입 현황, 권한, 홈페이지 수 제한 관리

    홈페이지 관리: 전체 리스트, 상태(활성/정지) 관리

    템플릿 관리: 등록/수정/삭제, AI 기반 템플릿 자동화

    시스템 통계: 전체 방문자 수, 퍼블리싱 건수, 리소스 사용량

    로그 모니터링: 사용자 활동, 에러 로그

3.5 각 홈페이지 관리자 페이지 (Site Admin)

    콘텐츠 관리: 게시글, 상품, 이미지, 데이터 관리

    사이트 설정: 로고, 메뉴, 컬러, SEO, SNS 연동

    방문자 통계: 페이지 뷰, 유입 경로 (GA4 연동 가능)

    사용자 관리(선택): 회원가입 기능을 사용하는 사이트라면 회원 리스트 및 권한 관리

    퍼블리싱 제어: 개별 페이지/전체 사이트 퍼블리싱

4. 비기능 요구사항

    확장성: 수천 명의 회원, 수만 개의 홈페이지를 동시에 서비스 가능

    성능: CDN 캐시 기반 빠른 응답

    보안: JWT 인증, 권한 레벨 구분 (User / Site Admin / Super Admin)

    가용성: AWS S3 + CloudFront 활용, 99.9% 이상 SLA 목표

    유지보수성: Headless 구조로 프런트엔드와 백엔드 완전 분리

5. 기술 스택 제안

    프런트엔드: React/Vue + GrapesJS

    백엔드 API: Node.js (NestJS) 또는 Python (FastAPI)

    데이터베이스: MySQL/PostgreSQL (회원·홈페이지·설정 정보 관리)

    스토리지: AWS S3 + CloudFront

    인증: JWT + OAuth

    통계: GA4, 자체 로그 기반 분석

6. 향후 확장 고려사항

    홈페이지 템플릿 마켓플레이스 (회원이 만든 템플릿 공유/판매)

    플러그인/위젯 시스템 (쇼핑몰, 예약, 결제, 챗봇 등)

    결제 구독 시스템(Stripe, 아임포트) → 회원별 홈페이지 한도 자동 변경

    다국어 지원(i18n)

📌 정리하면:

Super Admin(회사 전체 관리자 페이지): 전체 회원과 홈페이지, 리소스, 템플릿 관리

Site Admin(각 홈페이지 관리자 페이지): 개별 홈페이지 콘텐츠·설정·사용자 관리
